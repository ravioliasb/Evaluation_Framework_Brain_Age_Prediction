# Evaluation Framework for Brain Age Prediction

An evaluation framework for global and voxel-wise brain age prediction models.

* **Models:** Training and testing code for models SFCN, VOX1, and VOX2 found in `/Models`.
* **Experimental Setup:** Experimental code for Scenarios 1, 2, and 3 found in `/Experimental_Setup`.
* **Synthetic Data Generation:** Find the code at [wilmsm/lightweightbraindiff](https://github.com/wilmsm/lightweightbraindiff).

Email ravi.bullock@ucalgary.ca for additional information.
